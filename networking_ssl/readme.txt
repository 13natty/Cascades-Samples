
networking_ssl

========================================================================
Sample Description.

The networking_ssl example shows you how to create a secure SSL connection to a remote server, and send or receive data.

You will learn how to:
 - Set up your project
 - Set up a secure connection that uses SSL and port 443
 - Initiate a secure connection and verify that it succeeded
 - Retrieve certificate information, which can be used for authentication, from the remote server
 - Send an HTTPS request over a network
 - Retrieve the network reply data
 - Communicate between C++ and QML

========================================================================
Requirements:

BlackBerry 10 Native SDK

========================================================================
Running the example:

1. Clone the Sample repository.
2. Launch the Momentics IDE for BlackBerry, and from the File menu, select Import.
3. Expand General, and select Existing Projects into Workspace. Click Next.
4. Browse to the location of your sample directory, and then click OK.
5. The sample project should display in the Projects section.
   Click Finish to import the project into your workspace.
6. In the Project Explorer pane, Right-click the project (for example networking_ssl)
   and select Build Project.
7. In the Project Explorer pane, Right-click the project (for example networking_ssl)
   and select Run As > BlackBerry C/C++ Application.
8. The application will now install and launch on your device. If not, you might
   have to set up your environment:
   http://developer.blackberry.com/native/documentation/getting_started/setting_up.html
