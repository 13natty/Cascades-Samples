Push Collector Common

========================================================================
Description.

The Push Collector Common is a shared library that is packaged with the
Push Collector application. It is used by both the UI and headless part 
of the Push Collector application. 

The sample code for this library is Open Source under the Apache 2.0 License
(http://www.apache.org/licenses/LICENSE-2.0.html).

========================================================================
Requirements:

BlackBerry 10.2 Native SDK
